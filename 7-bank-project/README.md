# :dollar: 构建银行应用

在这个项目中，您将学习如何构建一个虚构的银行应用。这些课程包括有关如何布局 Web 应用程序并提供路由、构建表单、管理状态以及从 API 中获取数据的说明，您可以从中获取银行数据。
