# 练习你的 HTML：搭一个博客版式草图

## 说明

假设你正在设计（或重新设计）你的个人网站。先创建你网站的视觉草图模型，然后写下你会用来构建网站各个元素的 HTML 标记。你可以在纸上完成并扫描，或使用你喜欢的软件，但务必手写 HTML 标记（不要用可视化生成器）。

| -------- | ----------------------------------------------------------------------------------- | ----------------------------------------------------------------------------- | -------------------------------------------------------------------------------- |
